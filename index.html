Guia de Uso do Fogão Acessível

Manual de Utilização do Fogão

Este fogão foi projetado para ser seguro e fácil de usar por todas as pessoas, incluindo idosos e pessoas com deficiência visual. Todos os botões têm texturas diferentes e emitem sons para confirmar seus comandos.

Explicação dos Botões

Botões 1 a 4

Controlam as bocas do fogão (1 = frente esquerda, 2 = frente direita, 3 = trás esquerda, 4 = trás direita)

Botões 5 a 8

Controlam a temperatura (5 = mínimo, 6 = baixo, 7 = médio, 8 = alto)

Botão 9

Liga/desliga o fogão (pressione por 3 segundos)

Botão 10

Bloqueio de segurança para crianças

Como Acender uma Boca do Fogão

Passo a Passo:

  1. Pressione o Botão 9 por 3 segundos até ouvir um bip longo
  2. Toque no Botão 1, 2, 3 ou 4 para selecionar a boca desejada
  3. O fogão emitirá um som confirmando qual boca foi selecionada
  4. Pressione um dos Botões 5 a 8 para escolher a temperatura
  5. Você ouvirá um bip para cada nível de temperatura selecionado

Como Ajustar a Temperatura

Escala de Temperatura:

  • Botão 5 (Mínimo): Para manter alimentos aquecidos
  • Botão 6 (Baixo): Para derreter e cozinhar lentamente
  • Botão 7 (Médio): Para cozinhar a maioria dos alimentos
  • Botão 8 (Alto): Para ferver água e fritar

Dica: Cada mudança de temperatura é confirmada com um som diferente!

Dicas de Segurança

!

Sempre use cabos de panelas virados para dentro
